## Cykl Życia Komponentu

```
┌─────────────────────────────────────────────────────────────────┐
│                   LIFECYCLE: StaffBottomBar                     │
└─────────────────────────────────────────────────────────────────┘

1. INICJALIZACJA
   ┌────────────────────────────────────────┐
   │ HomeActivity.onCreate()                │
   │   └─> remember { mutableIntStateOf(0) }│
   └────────────────┬───────────────────────┘
                    │
                    ▼
2. KOMPOZYCJA
   ┌────────────────────────────────────────┐
   │ @Composable StaffBottomBar()           │
   │   └─> NavigationBar { ... }            │
   └────────────────┬───────────────────────┘
                    │
                    ▼
3. RENDEROWANIE
   ┌────────────────────────────────────────┐
   │ staffTabs.forEachIndexed { ... }       │
   │   └─> NavigationBarItem(...)           │
   └────────────────┬───────────────────────┘
                    │
                    ▼
4. INTERAKCJA
   ┌────────────────────────────────────────┐
   │ onClick = { onTabSelected(index) }     │
   │   └─> selectedStaffTabIndex = index    │
   └────────────────┬───────────────────────┘
                    │
                    ▼
5. REKOMPOSYCJA
   ┌────────────────────────────────────────┐
   │ Stan zmieniony → rekomposycja          │
   │   └─> Nowe podświetlenie zakładki      │
   └────────────────┬───────────────────────┘
                    │
                    ▼
6. SYNCHRONIZACJA
   ┌────────────────────────────────────────┐
   │ StaffView.LaunchedEffect               │
   │   └─> pagerState.animateScrollToPage() │
   └────────────────────────────────────────┘
```
